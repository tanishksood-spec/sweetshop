# Hadimba Sweets – Desktop POS System

A beautiful, fully functional **Point of Sale (POS) system** built specially for **Hadimba Sweets**, Manali – a traditional Indian sweet shop. Made with Python and CustomTkinter, this app lets you sell sweets faster, generate professional bills instantly, and manage orders with zero hassle.

### Features
- Stunning product grid with real photos of Gulab Jamun, Rasmalai, Kaju Katli, etc.
- Live search & category filter (Bengali, Milk Sweets, Dry Fruits, etc.)
- Smart shopping cart with quantity editing and remove option
- Customer details collection (Name, Phone, Address)
- Professional PDF bill generation with your shop name & logo
- Auto-opens the bill instantly after checkout (no need to search folder)
- All orders & items saved in local SQLite database
- Receipts automatically saved in `receipts/` folder
- 100% offline – works without internet
- Zero installation for end user (can be converted to .exe later)

### Perfect For
- Hadimba Sweets, Manali
- Any Indian sweet shop / mithai store
- Small bakery or namkeen counter

### Tech Stack
- Python 3
- CustomTkinter (modern beautiful UI)
- PIL/Pillow (image handling)
- ReportLab (PDF generation)
- SQLite (lightweight database)
